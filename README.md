## Установка и запуск

Сначала нужно клонировать репозиторий
```bash
git clone https://github.com/separeit894/torrc_setup
```

### Сборка скрипта

#### Можно собрать скрипт через Makefile внутри папки
```bash
make
```

#### Через g++
```bash
g++ -o torrc_setup.exe main.cpp create_file_config_and_read/CreateFileConfigAndRead.cpp read_file/ReadFile.cpp set_path_to_torrc/SetPathToTorrc.cpp write_file/WriteFile.cpp create_backup_torrc_file/CreateBackupTorrcFile.cpp
```

#### Через clang++
```bash
clang++ -o torrc_setup.exe main.cpp create_file_config_and_read/CreateFileConfigAndRead.cpp read_file/ReadFile.cpp set_path_to_torrc/SetPathToTorrc.cpp write_file/WriteFile.cpp create_backup_torrc_file/CreateBackupTorrcFile.cpp
```

### Запуск скрипта и как им пользоваться

#### Запуск скрипта

```bash
torrc_setup.exe
```
или на Linux

```bash
./torrc_setup
```

#### Как пользоваться скриптом

При запуске у вас попросит ввести путь к файлу torrc
Вы можете ввести абсолютный путь
```bash
C:\Полный\путь\до\файла\torrc
```

Или относительный путь 

```bash
Папка\torrc
```

После того как вы ввели расположение файла torrc, вас попросят ввести новые мосты
Вы их вставляете в программу, она их добавит. Дальше вы проверяете сработала ли замена моста. Удачного использования скрипта
